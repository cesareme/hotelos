# SES.HOSPEDAJES Official Schema Placeholder

HotelOS does not hardcode unofficial SES.HOSPEDAJES XML, TXT or web-service payload schemas.

Do not hardcode unofficial schemas in this folder or in adapters.

The exact official schema, XSD, template or service-web documentation must be downloaded from the official SES.HOSPEDAJES platform by the hotel or admin user, then configured in Back Office before production batch generation is enabled.

This folder is intentionally reserved for:

- Official XSD or template references supplied by the authority.
- Schema version metadata.
- Adapter validation notes.
- Hotel-specific service-web activation evidence.

Until an official schema is configured, the platform can generate preview payloads and manual mock exports, but production submission remains blocked.
