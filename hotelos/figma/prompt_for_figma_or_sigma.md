# Prompt For Figma Or Sigma

Design a futuristic, intuitive, mobile-first hotel operating system called HotelOS.

Do not copy existing PMS products. Create an original design language: premium hospitality plus operational command center plus AI-native interface.

Core principles:
- Mobile-first, tablet-enhanced, admin-web capable.
- AI is a first-class interface, not a small chatbot.
- Use cards, bottom sheets, command dock, contextual action panels, status chips and confidence indicators.
- Use progressive disclosure.
- Every critical action uses a confirmation card.
- Every module has clear health state and setup state.
- Accessibility: large tap targets, high contrast, clear focus states, readable text.

Visual direction:
- Warm light operational surfaces.
- Night blue and deep indigo primary.
- Violet and electric blue AI accent.
- Aqua success highlight.
- Amber warning.
- Red critical.
- Rounded cards, soft shadows, precise spacing.
- No overloaded tables on mobile.

Create these screens:
1. Today Dashboard
2. AI Command Center
3. Voice check-in flow: "Check-in this customer in room 432"
4. Temporary ID scan plus OCR confidence fields
5. Check-in Confirmation Card
6. Rooms list with micro timeline
7. Tablet PMS planning
8. Housekeeping board
9. Maintenance board
10. Compliance Inbox
11. Owner Dashboard
12. Back Office Property Mapper
13. Module Marketplace
14. Integration Marketplace
15. AI Governance screen
16. Guest portal online check-in

Use token names from `design-tokens/hotelos.tokens.json`.
